### **Toggleable Fog Types**:
- **Terrain Fog**: Remove the fog that depends on the drawing distance, which hides the boundaries of the chunks.
- **Atmospheric Fog**: Remove sky fog in the Overworld and thick fog in the Nether.
- **Lava Fog**: Remove fog in lava for clearer vision.
- **Water Fog**: Clear underwater fog for better visibility.
- **Powder Snow Fog**: Eliminate fog effects in powder snow.
- **Blindness Fog**: Disable fog caused by the blindness effect.
- **Darkness Fog**: Remove fog from the darkness effect (e.g., from wardens).

---

By default, the mod disables terrain fog and atmospheric fog. All other fog types can be configured individually, including per-dimension overrides.

### Configuration:

Starting from version 2.0.0, the mod includes a built-in configuration screen available on all mod loaders — no additional mods required.

### For Fabric and Quilt:
- Install [ModMenu](https://modrinth.com/mod/modmenu) to access the configuration screen in-game.

### For Forge and NeoForge:
- No additional mods required. Select the mod from the mod list and click the "Config" button.

### Per-Dimension Settings:

Each dimension can independently override the global fog settings. If a dimension setting is set to DEFAULT, it inherits the value from the global configuration. Dimensions added by other mods are supported automatically.

---

### ⚠️ Note for versions below 2.0.0:

Built-in configuration is available starting from version 2.0.0. On older versions, install [ClothConfig](https://modrinth.com/mod/cloth-config) and [ModMenu](https://modrinth.com/mod/modmenu) to configure the mod in-game.
